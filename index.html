Web 前端必备利器

掌握以下技术，提升开发能力，做一名优秀的Web前端工程师。

Css框架 Bootstrap

Bootstrap 是最受欢迎的 HTML、CSS 和 JS 框架，用于开发响应式布局、移动设备优先的 WEB 项目。

为所有开发者、所有应用场景而设计。 Bootstrap 让前端开发更快速、简单。所有开发者都能快速上手、所有设备都可以适配、所有项目都适用。 中文网站

预处理脚本 Less

Less 是一门 CSS 预处理语言，它扩充了 CSS 语言，增加了诸如变量、混合（mixin）、函数等功能，让 CSS 更易维护、方便制作主题、扩充。 中文网址为 http://less.bootcss.com

Less 可以运行在 Node、浏览器和 Rhino 平台上。网上有很多第三方工具帮助你编译 Less 源码。

Javascript 平台 Node.js

Node.js是一个基于Chrome JavaScript运行时建立的平台， 用于方便地搭建响应速度快、易于扩展的网络应用。 Node.js 使用事件驱动， 非阻塞I/O 模型而得以轻量和高效，非常适合在分布式设备上运行的数据密集型的实时应用。 Node.js 包管理器 npm,the nodejs package manager

Web前端管理包工具 Bower

Bower 是 twitter 推出的一款前端包（Web包通常认为由html+css+javascript构成）管理工具，基于Node.js的模块化思想， 把功能分散到各个模块中，让模块和模块之间存在联系，通过 Bower 来管理模块间的联系。

JavaScript 构建工具 Grunt

对于需要反复重复的任务，例如压缩（minification）、编译、单元测试、linting等，自动化工具可以减轻你的劳动，简化你的工作。 当你在 Gruntfile 文件正确配置好了任务，任务运行器就会自动帮你或你的小组完成大部分无聊的工作。 中文网址

Web应用脚手架工具 Yeoman

Yeoman是Google的团队和外部贡献者团队合作开发的，他的目标是通过Grunt和Bower的包装为开发者创建一个易用的工作流。

Yeoman主要有三部分组成：yo（脚手架工具）、grunt（构建工具）、bower（包管理器）。 这三个工具是分别独立开发的，但是需要配合使用，来实现我们高效的工作流模式。
