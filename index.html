Web 前端必备利器

掌握以下技术，提升开发技能，做一名优秀的Web前端工程师。

优秀的Javascript库 jQuery

jQuery是一个兼容多浏览器的javascript库，核心理念是write less,do more(写得更少,做得更多)。 jQuery在2006年1月由美国人John Resig在纽约的barcamp发布，吸引了来自世界各地众多JavaScript高手加入，由Dave Methvin率领团队进行开发。 如今，jQuery已经成为最流行的Javascript库，在世界前10000个访问最多的网站中，有超过55%在使用jQuery。

优秀web前端 MVC 框架 Angular

Angular 是用来构建手机和PC端 Web 应用的一款非常优秀的开发平台。它的创新点包括数据双向绑定、依赖注入、指令、MVC和模板等。 Angular 是一个为动态 Web 应用设计的前端框架，创新地把后台技术融入到前端开发。 让你通过扩展HTML的语法，能更清楚、更规范、更结构化、更可控、更简洁地构建Web应用。 中文网 中文社区 Angular UI

轻量级MVVM框架 KnockoutJS

KnockoutJS，简称ko，是一个轻量级的MVVM框架，通过简易的UI绑定语法，实现动态更新UI。其特点有： 1.声明式绑定：通过简洁易读的data-bind语法，将DOM元素与ViewModel关联起来。 2.UI自动更新：当模型（ViewModel）状态更新时，自动更新UI界面。 3.依赖跟踪：在模型数据间建立隐式的关系链。 4.模板化：快速生成复杂，可嵌套UI模型数据的函数。 中文参考资料

CSS 框架 Bootstrap

Bootstrap 是最受欢迎的 HTML、CSS 和 JS 框架，用于开发响应式布局、移动设备优先的 WEB 项目。

为所有开发者、所有应用场景而设计。 Bootstrap 让前端开发更快速、简单。所有开发者都能快速上手、所有设备都可以适配、所有项目都适用。 中文网站

预处理脚本 Less

Less 是一门 CSS 预处理语言，它扩充了 CSS 语言，增加了诸如变量、混合（mixin）、函数等功能，让 CSS 更易维护、方便制作主题、扩充。 中文网址为 http://less.bootcss.com

Less 可以运行在 Node、浏览器和 Rhino 平台上。网上有很多第三方工具帮助你编译 Less 源码。在线Less编辑器 less2css.org

完美的图标字体库 Font Awesome

Font Awesome 向你提供可缩放向量图标， 你可以即刻进行个性化 — 大小，颜色，阴影效果，以及任何可以用CSS的力量解决的事情。 中文网

Font Awesome gives you scalable vector icons that can instantly be customized — size, color, drop shadow, and anything that can be done with the power of CSS.

Web前端管理包工具 Bower

Bower 是 twitter 推出的一款前端包（Web包通常认为由html+css+javascript构成）管理工具，基于Node.js的模块化思想， 把功能分散到各个模块中，让模块和模块之间存在联系，通过 Bower 来管理模块间的联系。

JavaScript 构建工具 Grunt

对于需要反复重复的任务，例如压缩（minification）、编译、单元测试、linting等，自动化工具可以减轻你的劳动，简化你的工作。 当你在 Gruntfile 文件正确配置好了任务，任务运行器就会自动帮你或你的小组完成大部分无聊的工作。 中文网址

Web应用脚手架工具 Yeoman

Yeoman是Google的团队和外部贡献者团队合作开发的，他的目标是通过Grunt和Bower的包装为开发者创建一个易用的工作流。

Yeoman主要有三部分组成：yo（脚手架工具）、grunt（构建工具）、bower（包管理器）。 这三个工具是分别独立开发的，但是需要配合使用，来实现我们高效的工作流模式。

Javascript 平台 Node.js

Node.js是一个基于Chrome JavaScript运行时建立的平台， 用于方便地搭建响应速度快、易于扩展的网络应用。 Node.js 使用事件驱动， 非阻塞I/O 模型而得以轻量和高效，非常适合在分布式设备上运行的数据密集型的实时应用。 Node.js 包管理器 npm The nodejs package manager

CSS的开发工具 Sass

Sass 扩展了 CSS3，增加了规则、变量、混入、选择器、继承等等特性。Sass 生成良好格式化的 CSS 代码，易于组织和维护。 SASS是对CSS3（层叠样式表）的语法的一种扩充，它可以使用巢状、混入、选择子继承等功能，可以更有效有弹性的写出Stylesheet。 中文网址 入门教材

自动化构建工具 Gulp

  • 易于使用：通过代码优于配置的策略，Gulp 让简单的任务简单，复杂的任务可管理。
  • 构建快速：利用 Node.js 流的威力，你可以快速构建项目并减少频繁的 IO 操作。
  • 插件高质：Gulp 严格的插件指南确保插件如你期望的那样简洁高质得工作。
  • 易于学习：通过最少的 API，掌握Gulp毫不费力，构建工作尽在掌握，如同一系列流管道。
